Pozitia!

Statica = valoare implicita, pozitia elementului este cea normala.

Relativa = elementul este mutat de la pozitia normala intr-o directie(top, bottom,right,left)

fixata = pozitia va ramane aceeasi, si daca se trece de acea parte a paginii

Absoluta = asemenator cu pozitia relativa, numai ca elementul este pozitionat dupa primul element parinte care este pozitionat(are pozitia diferita de cea statica)

Sticky = elementul va oscila intre pozitia relativa si cea fixata, daca se trece de partea lui, atunci elementul va avea pozitia fixata.
