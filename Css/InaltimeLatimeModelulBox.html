Acest div are inaltimea 400px si latimea de 1200px
Max-width si Max-height sunt utile atunci cand latimea/lungimea browserului este mica.

Inaltimea efectiva = height + padding-top + padding-bottom + margin-top + margin-bottom +border-top-width+border-bottom-width

Lungimea efectiva = width + padding-right + padding-left + margin-right+margin-left+border-right-width+border-left-width
